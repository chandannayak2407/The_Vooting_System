# 🗳️ Simple Voting System (Java + MySQL)

A basic console-based voting system developed using **Java (JDBC)** and **MySQL**.  
This project allows users to view candidates, cast votes, and see results.

---

## 🚀 Features

- 📋 Display list of candidates  
- 🗳️ Cast vote for a candidate  
- 📊 View voting results  
- ❌ No login or signup required (simple version)  
- 💻 Console-based application  

---

## 🛠️ Technologies Used

- Java (Core Java)
- JDBC (Java Database Connectivity)
- MySQL Database
- VS Code
